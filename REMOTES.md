# Remote Repositories

The project uses the following synchronized upstream git remotes:
1. **GitHub:** [https://github.com/funbinet/jabber](https://github.com/funbinet/jabber)
2. **Codeberg:** [https://codeberg.org/funbinet/jabber](https://codeberg.org/funbinet/jabber)

All deployments and tagging should simultaneously interact with both repositories to ensure platform integrity and parity.
